1. O QUE É O SELENIUM?

- Selenium is a free and open source library for automated testing web applications;
- Selenium supports all major browsers (Firefox, Chrome, IE);
- tests can be written in many programming languages (Java, C#, Ruby, Python) and run on Windows, Linux and Macintosh platforms;
