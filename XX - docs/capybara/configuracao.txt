::PREPARANDO O SISTEMA::

É preciso instalar as seguintes Gems:
- Ruby (https://www.ruby-lang.org/pt/)
- Bundler (http://bundler.io/)
- Nokogiri (http://www.nokogiri.org/tutorials/installing_nokogiri.html)
- Capybara (http://jnicklas.github.io/capybara/)
- Cucumber (https://cucumber.io/)
- Selenium Webdriver (http://www.seleniumhq.org/projects/webdriver/)
- Cucumber Rails (https://github.com/cucumber/cucumber-rails)


1. Instalar a versão mais atualizada do Ruby:

a) gem install ruby (instala a versão mais autalizada do Ruby)
b) ruby -v (verifica a versão mais atualizada do Ruby)
c) gem -v (verifica a versão mais atualizada das Gems)
d) gem update --system (atualiza todas as gems do sistema)

2. Instalar o Bundler:

a) gem install bundler (instala a versão mais autalizada do Bundler)
b) bundle install (instala gems específicas para o projeto selecionado)
c) bundle exec cucumber (gera as etapas/snippets adaptadas para Ruby criadas previamente no arquivo de feature)

3. Instalar Nokogiri:

a) gem install nokogiri (instala a versão mais autalizada do Nokogiri)

4. Em caso de problemas... execute os seguintes comandos:

sudo apt-get install build-essential patch (Compila todos os pacotes/extensões em C)
sudo apt-get install ruby-dev zlib1g-dev liblzma-dev (Instala todas as gems pendentes)

5. Instalar Capybara:

a) gem install capybara

6. Instalar Cucumber:

a) gem install cucumber

7. Instalar Selenium Webdriver:

a) gem install selenium-webdriver

8. Instalar Cucumber Rails:

a) gem install cucumber-rails
b) cucumber (Executa o script de teste)
